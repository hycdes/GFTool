Linux/terminal_basis

基本terminal指令

获取root权限

    sudo passwd # 首次设置root密码

    su # root模式可省略sudo

显示进程

    ps

    ps -A # 显示所有进程

    ps -A | grep abcd # 显示名字带"abcd"的进程

创建目录

    mkdir -p NAME # 创建NAME目录，-p确保目录存在否则创建

查看文件

    cat NAME

更新现有包

    sudo apt-get update

安装典型依赖包

    sudo apt-get install PACKNAME

    一些常用的包

        sudo apt-get install build-essential # 编译用

        sudo apt-get install net-tools # 网络工具

.gz文件

    tar -zxvf NAME.tar.gz # 解压.gz，输入名字一半按TAB可自动出全称

    tar -zcvf NEW_NAME.tar.gz NAME # 压缩成.gz

.xz文件

    xz -d NAME.tar.xz # 解压.xz为.tar

    tar -xvf NAME.tar # 解压.tar
